## 🎯 نظرة عامة

تم تطوير **Spacetime Explorer** ليصبح منصة شاملة لاستكشاف الفضاء والفيزياء تجمع بين:
- **التعليم التفاعلي** - دروس بصرية ومحاكاات
- **المحاكاات العلمية** - محاكاات دقيقة للظواهر الفيزيائية
- **الأدوات التقاعلية** - حاسبات وأدوات متخصصة
- **المجتمع العالمي** - مشاركة واستكشاف مشترك
- **البيانات الحقيقية** - بيانات مباشرة من وكالات الفضاء

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js 14+
- npm أو yarn

### خطوات التثبيت

```bash
# 1. انسخ المشروع
git clone https://github.com/turkiabdullah3-dev/Black-hole.git
cd Black-hole

# 2. ثبّت المتعلقات
npm install

# 3. شغّل خادم التطوير
npm run dev

# 4. افتح في المتصفح
# http://localhost:3000
```

### بناء الإصدار النهائي

```bash
# بناء الإصدار المُحسّن
npm run build

# معاينة الإصدار
npm run preview
```

## 👨‍💻 دليل المطورين

### إضافة ميزة جديدة

1. **إنشاء صفحة HTML**
```html
<!-- src/pages/new-feature.html -->
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>الميزة الجديدة</title>
    <link rel="stylesheet" href="../styles/new-feature.css">
</head>
<body>
    <!-- المحتوى -->
    <script type="module" src="../utils/new-feature.js"></script>
</body>
</html>
```

2. **إنشاء ملف CSS**
```css
/* src/styles/new-feature.css */
@import url('./common.css');

/* أضف أنماطك هنا */
```

3. **إنشاء ملف JavaScript**
```javascript
// src/utils/new-feature.js
// أضف منطقك هنا
```

4. **تحديث الملاح**
```html
<!-- أضف رابط في home.html -->
<a href="src/pages/new-feature.html">الميزة الجديدة</a>
```

### الألوان المتاحة

--primary-color: #00ffdc;      /* أزرق فاتح */
