## 🧮 معادلات الفيزياء المدرجة

### المعادلات الأساسية:

#### 1. نصف قطر شفارتسشيلد
```
r_s = 2GM/c² 
    = 3(M/M☉) كم
```
**الاستخدام:** حساب حجم الثقب الأسود المحتمل من كتلة معينة

#### 2. سرعة الهروب
```
v_e = √(2GM/r)
```
**الاستخدام:** سرعة الهروب من سطح أي جسم سماوي

#### 3. تمدد الزمن النسبوي (معامل لورنتز)
```
γ = 1/√(1 - v²/c²)
```
**الاستخدام:** حساب تمدد الزمن عند السفر بسرعات عالية

#### 4. الانزياح الأحمر النسبوي
```
z = √((1+β)/(1-β)) - 1
   حيث β = v/c
```
**الاستخدام:** تأثير دوبلر على الضوء من الأجسام المتحركة

#### 5. قوة المد والجزر
```
F = 2GMm·d/r³
```
**الاستخدام:** القوة التي تمزق الأجسام بسبب فروق الجاذبية

#### 6. مدار ISCO
```
r_isco = 6r_s = 12GM/c²
```
**الاستخدام:** أقرب مدار مستقر حول ثقب أسود

#### 7. السرعة المدارية
```
v = √(GM/r)
```
**الاستخدام:** حساب سرعة دوران الأجسام حول جسم أكبر

### الثوابت المستخدمة (من CODATA 2018):
- **G** = 6.67430e-11 m³ kg⁻¹ s⁻²
- **c** = 299,792,458 m/s
- **M_sun** = 1.989e30 kg
- **M_earth** = 5.972e24 kg
- **R_earth** = 6,371,000 m
- **AU** = 1.496e11 m
- **h** = 6.62607015e-34 J·s
